# 182. 查找重复的电子邮箱（简单）#【sql】

标签（空格分隔）： in_order

---
编写一个 SQL 查询，查找 Person 表中所有重复的电子邮箱。

示例：

    +----+---------+
    | Id | Email   |
    +----+---------+
    | 1  | a@b.com |
    | 2  | c@d.com |
    | 3  | a@b.com |
    +----+---------+

根据以上输入，你的查询应返回以下结果：

    +---------+
    | Email   |
    +---------+
    | a@b.com |
    +---------+

### 我的做法：  
显然不会啊，多少年没写sql了- -  
```sql
select Email from Person
group by Email having count(Email) > 1;
```
