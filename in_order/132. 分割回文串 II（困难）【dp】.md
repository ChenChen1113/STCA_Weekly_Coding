# 132. 分割回文串 II（困难）【dp】

标签（空格分隔）： in_order

---
给定一个字符串 s，将 s 分割成一些子串，使每个子串都是回文串。

返回符合要求的最少分割次数。

示例:

    输入: "aab"
    输出: 1
    解释: 进行一次分割就可将 s 分割成 ["aa","b"] 这样两个回文子串。


### 我的做法：  
回溯果然超时……  显然要用dp    方程：
```C++
if(isPalindrome(s, 0, i)){
    dp[i] = 0;
}else{
    dp[i] = min(dp[j] + 1, dp[i]);
}
```
```C++
class Solution {
public:
    int minCut(string s) {
        if(s.size() == 0){return 0;}
        vector<int> dp(s.size());
        dp[0] = 0;
        for(int i = 1; i < s.size(); i++){
            if(isPalindrome(s, 0, i)){
                dp[i] = 0;
            }else{
                dp[i] = INT_MAX;
                for(int j = 0; j < i; j++){
                    if(isPalindrome(s, j+1, i)){
                        dp[i] = min(dp[j] + 1, dp[i]);
                    }
                }
            }
        }
        return dp[s.size() - 1];
    }

    bool isPalindrome(string& s, int start, int end){
        while(start <= end){
            if(s[start] != s[end]){
                return false;
            }
            start ++;
            end --;
        }
        return true;
    }
};
```
