# 171. Excel表列序号（简单）

标签（空格分隔）： in_order

---
给定一个Excel表格中的列名称，返回其相应的列序号。

例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...

示例 1:

    输入: "A"
    输出: 1

示例 2:

    输入: "AB"
    输出: 28

示例 3:

    输入: "ZY"
    输出: 701

### 我的做法：  
26进制转10进制
```C++
class Solution {
public:
    int titleToNumber(string s) {
        int res = 0;
        for(int i = 0; i < s.size(); i++){
            int cur = s[i] - 'A' + 1;
            res = res * 26 + cur;
        }
        return res;
    }
};
```
