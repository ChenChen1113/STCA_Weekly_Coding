# 168. Excel表列名称（简单）

标签（空格分隔）： in_order

---
给定一个正整数，返回它在 Excel 表中相对应的列名称。

例如，

    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...

示例 1:

    输入: 1
    输出: "A"

示例 2:

    输入: 28
    输出: "AB"

示例 3:

    输入: 701
    输出: "ZY"

### 我的做法：  
10进制转26进制
```C++
class Solution {
public:
    string convertToTitle(int n) {
        string res = "";
        while(n > 0){
            n -= 1;
            char c = 'A' + n%26;
            res += c;
            n /= 26;
        }
        string s(res.rbegin(), res.rend());
        return s;
    }
};
```
